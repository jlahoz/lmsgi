Caracteristicas:

  • Intel® GbE LAN con cFosSpeed Internet Accelerator Redes: Intel® GbE LAN cuenta con cFosSpeed, una aplicación de gestión de tráfico de red que ayuda
    a mejorar la latencia de la red y mantener tiempos de ping bajos para ofrecer una mejor capacidad de respuesta en entornos de LAN atestados.
  • Solución Fast USB 3.1 de segunda generación: El último UBS 3.1 de segunda generación ofrece velocidades de transferencia de 10Gb/s dos vecs más rápido
    que la anterior generación del USB 3.1, con compatibilidad con todos los dispositivos existentes USB.
  • Amplificador de cascos inteligente: Detecta automáticamente la impedancia del head-worn de tu dispositivo de audio, previene problemas como el volumen
    bajo y la distorsión para aquellos auriculares internos o cascos de máxima calidad que produzcan óptimas dinámicas de audio.
  • Compatible con tiras de luz RGBW y -UV LED:: Programable de manera independiente con el conector para las tiras de RGBW/-UV LED. ¿No es lo suficientemente
    brillante? Puedes instalar un RGB externo de 12V estándar, tiras de la luz de RGBW o RGB/UV al conector RGBW y controlar su color desde el software RGB Fusion.
    Podrás obtener colores más brillantes y vibrantes si utilizas una tira de luz de RGBW con un cuidado LED blanco. Al usar tiras de RGB/UV LED, los usuarios pueden
    dar a sus sistemas un resplandor fluorescente con el uso del líquido UV reactive, los cables trenzados de neón, y la pintura acrílica para un ordenador gaming que atraerá
    todas las miradas. No habrá necesidad de elegir la luz de las tiras ya que las placas base de AORUS soportan una gran variedad de tiras RGBW/-UV del mercado, gracias
    al software RGB Fusion que permite redefinir la ubicación de las clavijas para la señal de R/G/B.
  • Smart Fan 5: Con Smart Fan 5 los usuarios pueden asegurarse de que su PC gaming puede mantener un rendimiento estable mientras se mantiene refrigerado. Smart Fan 5
    permite a los usuarios intercambiar sus conectores para los ventiladores para reflejar diferentes sensores térmicos en diferentes ubicaciones de la placa base. Además con
    Smart Fan 5 los conectores híbridos para el ventilador soportan ventiladores PWM y de voltaje, esto hace que la placa base este más adaptada a montar refrigeración líquida.
  • Nuevo Software Smart Fan 5: Elige entre diferentes modos, desde silencioso hasta máxima velocidad.
  • Para cada ventilador o bomba, se puede utilizar la curva intuitiva del para definir los umbrales de disparo y las velocidades del ventilador correspondiente.
  • Curva del ventilador: Personaliza las curvas de los ventiladores en función de las temperaturas específicas con la interfaz gráfica del usuario.
  • Sensores y ventiladores interoperables: Cada ventilador puede personalizarse para operar a diferentes velocidades según la entrada de temperatura definida (excepto el ventilador
    de la CPU)
  • Blindaje de memorias Ultra Durable™: La exclusiva pieza de protección de acero inoxidable de AORUS previene de las deformaciones y alteraciones del PCB, redondeando la placa,
    además de prevenir cualquier interferencia ESD posible.
  • TriDef® SmartCam | Stream personalizado: Para facilitar la solución de streaming completa cada placa base AORUS incluye software TriDef® SmartCam, capaces de eliminar fondos
    sin la molestia de usar una pantalla verde. Gracias a SmartCam se podrán utilizar filtros vivos y controlar por completo el fondo, no sólo durante el stream, si no también con otras
    aplicaciones del vídeo cómo Skype.
